Detailed Description

second pass of the assembling

The goal is to complete the tables and create the full object code and link objects.

Simplified algorithm:

  1. instruction counter(IC) = 0
  2. read line
  3. if no lines left, done
  4. process column:
    • label, get next column, goto 4
    • .data/.string/.entry/.extern: goto 2
    • instruction:
      1. if operand is symbol, get value from symtable, complete object code at IC
      2. get the addressing mode, get length(L), add instruction word to object code, IC += L
  5. goto 2
